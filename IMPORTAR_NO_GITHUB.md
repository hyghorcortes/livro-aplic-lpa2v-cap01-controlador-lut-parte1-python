# Como subir este repositório ao GitHub

## Opção 1 — via interface web

1. Crie um novo repositório no GitHub com o nome:

```text
livro-aplic-lpa2v-cap01-controlador-lut-parte1-python
```

2. Extraia o conteúdo deste ZIP no seu computador.
3. Arraste todos os arquivos para o repositório recém-criado no GitHub.
4. Faça o commit inicial.

## Opção 2 — via git no terminal

```bash
git init
git branch -M main
git add .
git commit -m "feat: adiciona repositório do exemplo 1 - CP-LUT"
git remote add origin <URL_DO_SEU_REPOSITORIO>
git push -u origin main
```

## Observação

Se você quiser, depois pode preencher no livro os links diretos para:

- o repositório;
- o script principal;
- o pacote Python;
- a figura de preview.
