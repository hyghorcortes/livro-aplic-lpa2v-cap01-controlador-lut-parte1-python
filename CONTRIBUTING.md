# Contribuindo

Sugestões bem-vindas para:

- ajustes de documentação;
- melhoria de nomes e comentários;
- inclusão de novos cenários de simulação;
- validação cruzada com outras versões do exemplo.

Fluxo sugerido:

1. criar uma branch;
2. fazer a alteração;
3. testar localmente;
4. abrir pull request com descrição objetiva.
